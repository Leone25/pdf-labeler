FAQ

What are PDF page lables?

Page lables in PDFs are a way to identify each page. They are the classic arabic numerals(1,2,3,4...), but also other types of numeration like roman numerals(I,II,III,IV...). This application allows to mix and match them, to reflect the content of the document and ease navigation.

What kind PDF page lables exist?

In the PDF specification, there are 6 built in styles for page lables:

  • Arabic numerals
  • Uppercase roman numerals
  • Lowercase roman numerals
  • Uppercase letters
  • Lowercase letters
  • Custom lable
In a PDF you can freely mix and match this to your liking. For example you could use a custom lable to make the cover/first page of your document, then uppercase roman numerals for the index, and lastly arabic numerals for the actual content of your document.

How do I use the website?

Once selected a document, you'll be presented with the list of already existing page lables, you can then create new page lables, select a syle, a prefix and a starting number. Once you are done you can click save and the modified document will be downloaded.

How safe is the website?

All documents get processed locally on your browser, no data gets sent to any server. You can check the code on Github.
